### Usage

#### `run.js`
```
Usage: node run.js <image> [type] -t [host] -c [keys]

Options:
  -h, --help  Show help                                                [boolean]
  -t, --host                          [default: "http://ai-api.easyar.com:8080"]
  -c, --keys                                              [default: "keys.json"]

copyright 2018, sightp.com
```

#### `example`
```
node run.js onehandheart.jpg -c keys.json -t http://ai-api.easyar.com:8080

```
